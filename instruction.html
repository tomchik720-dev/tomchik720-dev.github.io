инструкция по скачиванию

1. Установка нужных библиотек и архиватора
  • .NET Framework — установите последнюю версию dotnet-sdk-9.0.301-win-x64.exe. Запустите приложение и следуйте инструкциям.
  • Visual C++ Redistributable — установите последнюю версию vc_redist.x64.exe. Запустите приложение и следуйте инструкциям.
  • DirectX — установите последнюю версию dxwebsetup.exe. Запустите приложение и следуйте инструкциям.
  • WinRAR — установите последнюю версию (ссылка для загрузки).
2. Подготовка системы к первому запуску
  • Отключение Безопасность Windows: перейдите в параметры системы (WIN + I), откройте "Безопасность Windows", отключите параметр "Защита в реальном времени".
  • Окончательно отключите Безопасность Windows с помощью программы dControl: Google Drive — ссылка для загрузки (пароль 1234), Yandex Disk — ссылка для загрузки (пароль 1234), Mega — ссылка для загрузки.
  • Удаление сторонних антивирусов: для этого я рекомендую использовать утилиту Antivirus Removal Tool. Ссылка для загрузки — Antivirus_Removal_Tool.zip. Извлеките файл из архива в любое удобное для вас место. Откройте приложение, затем просканируйте систему на наличие антивирусов, для этого нажмите кнопку Search. После сканирования если антивирус был обнаружен, удалите его через встроенный деинсталлятор.
  • Отключение анти-чита других игр: откройте диспетчер задач и отключите из автозагрузки все анти-читы и лаунчеры.
  • Отключение оверлеев: закройте все оверлеи которые появляются при запуске игры (например Riva, Discord, FPS, Action, Bandicam и прочие). Исключение: Оверлей Nvidia, Epic Games.
  • Удалите reshade, redux и моды если используете.
